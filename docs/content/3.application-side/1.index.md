# Overview

On the application-side this module offers:
::list{type="success"}
- [`useSession` composable for user-data access, sign-in, sign-out and more](/application-side/session-access)
- [Creation of custom sign-in pages](/application-side/custom-sign-in-page)
- [Middleware to protect your application on the application side](/application-side/protecting-pages)
::

Application-side usage refers to any code like pages, components or composables that are part of the universal server- and client-side rendering of Nuxt, see more in the [glossary](/further-reading/glossary).
